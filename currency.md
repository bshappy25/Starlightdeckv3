# Currency System

## Careon (₢)

**Symbol:** ₢

**Name:** Careon

### Value and Usage

Careon is the primary currency used throughout the Starlightdeck system. It represents value in transactions, rewards, and economic exchanges within the game environment.

### Currency Symbol

The Careon symbol (₢) should be displayed before numerical values:
- ₢100
- ₢1,500
- ₢50.00

### Denominations

Careon can be expressed in whole numbers or decimals, depending on the transaction type and precision needed.
